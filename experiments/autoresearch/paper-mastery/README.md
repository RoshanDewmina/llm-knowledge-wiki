# Paper Mastery Autoresearch Pack

## Editable Target

- Primary: `~/.hermes/skills/research/research-paper-mastery/SKILL.md`
- Artifacts: `wiki/studies/papers/`, `wiki/studies/anki/`, and
  `experiments/papers/`

## Frozen Evaluator

Use `evaluator.md` to score whether a paper session creates durable mastery
artifacts instead of a passive summary.

## Procedure

1. Choose one paper section or one existing study note.
2. Run the current workflow without changing it.
3. Score the artifacts with the frozen evaluator.
4. Change one prompt/workflow instruction.
5. Repeat on a comparable paper section and append a ledger row.
