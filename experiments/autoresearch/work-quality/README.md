- `wiki/benchmarks/autoresearch/`

## Frozen Evaluator

Use `evaluator.md` to score one day from evidence, not vibes.

## Procedure

1. Create a morning plan in `wiki/journal/YYYY-MM-DD-plan.md`.
2. Log concrete work in `streak` with notes.
3. Create an evening review in `wiki/reviews/daily/YYYY-MM-DD-review.md`.
4. Score the day on output, depth, retention, shipping, focus, and goal alignment.
5. Weekly, pick one behavior experiment and append a result row.
