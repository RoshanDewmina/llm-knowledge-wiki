# derivation-coach

Use when the user wants to rebuild an equation or derivation.

Steps:
1. Scaffold or open `wiki/studies/derivations/<concept-slug>.md`.
2. Break the derivation into checkpoints: objects, dimensions, definitions, rewrite, invariant checks.
3. Ask the user to fill missing steps before revealing answers.
4. Link back to the concept and paper study.
5. Keep citations pending until exact evidence anchors exist.
