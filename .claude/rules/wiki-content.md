# Wiki Content Rules

- `raw/` is source material; `wiki/` is the maintained knowledge layer.
- Extend an existing page before creating a new near-duplicate.
- Reviewed source pages should add exact evidence anchors under `## Evidence Extracts` with `### ex-...` headings.
- Derived concept, synthesis, review, and non-slide output pages should include `## Citations` with exact source anchors when available.
- Keep an explicit `## Contradictions` section when the page type warrants it.
- If a new durable page is added, update `wiki/index.md` and `wiki/log.md`.
